- YouTube embeds
- Vimeo videos
- Product demos
- Walkthrough recordings

**Format:** `https://youtube.com/watch?v=abc | https://vimeo.com/123`

### 2. Multiple Media Per Project
- **Images:** Multiple screenshot URLs separated by `|`
- **Videos:** Multiple video URLs supported
- **Audio:** Multiple audio files for demos/recordings
- **Captions:** Individual captions for each media item

### 3. Easy Expansion
Simply copy-paste rows to add more:
- Projects
- Work experiences
- Skills
- Education entries

### 4. Built-in Validation
Status columns have dropdown: `Active | Hidden`
- Active = Shows on website
- Hidden = Temporarily hide without deleting

### 5. Display Order Control
Use `Display_Order` column (1, 2, 3...) to control:
- Which projects appear first
- Order of work experiences
- Skill presentation order
